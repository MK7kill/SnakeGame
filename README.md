# 🐍 SnakeGame — 贪吃蛇

一个使用原生 JavaScript + Canvas 构建的经典贪吃蛇游戏，带有多彩视觉效果和音效反馈。

## ✨ 特性

- 🎮 键盘方向键 / WASD 控制
- 📱 移动端触控按钮支持
- ⚡ 自适应加速（每 5 分升一级）
- 🌟 金色特殊食物（+3 分，15% 概率出现）
- 🔊 Web Audio API 音效（无需外部文件）
- 🖥️ 边界环绕（穿墙）
- 🏆 完美通关（占满画布）

## 🚀 快速开始

直接在浏览器打开 `index.html` 即可游玩。

## 🎯 操作

| 按键 | 功能 |
|------|------|
| ↑ ↓ ← → / WASD | 控制方向 |
| 空格键 | 暂停 / 继续 |
| 加速按钮 | 手动加速 |
